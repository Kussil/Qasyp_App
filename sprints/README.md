# Sprints

Sprint briefs, planning notes, and retrospectives for Qasyp App.

## Naming Convention
- `sprint-{N}-brief.md` — pre-sprint planning brief (generated by Sprint Prep Agent)
- `sprint-{N}-retro.md` — post-sprint retrospective notes

## Sprint Cadence
- Length: 2 weeks
- Ceremonies: Planning → Daily Standup → Review → Retrospective

## To generate a sprint brief
Ask Claude: _"Run the sprint prep agent for sprint {N}"_ and point it to `.claude/agents/sprint-prep/AGENT.md`.
